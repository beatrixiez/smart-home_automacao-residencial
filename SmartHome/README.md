### 🎯 Objetivo:

#### O site para a empresa fictícia de automação residencial SmartHome. Nele é possível conhecer mais do negócio, verificar os serviços oferecidos, consultar as marcas parceiras, tirar dúvidas frequentes e entrar em contato para obter mais informações.

### 🛠 Ferramentas:

#### Criado apenas com HTML5, CSS3, JavaScript e o framework W3.CSS.

### 👩🏻‍💻 Execução:

#### Baixar a pasta completa do repositório, extrair a pasta compactada e abrir o arquivo _index.html_ (dois cliques) em um navegador.
